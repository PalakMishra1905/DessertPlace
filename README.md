# 🍰 DessetPlace - Dessert Delivery Website

DessetPlace is a full-stack web application built with the **MERN stack (MongoDB, Express.js, React, Node.js)**. It offers users a seamless experience for browsing, ordering, and receiving delicious desserts at their doorstep.

---

## 🚀 Features

- 🍩 Browse a variety of desserts with images, prices, and descriptions
- 🛒 Add to cart and place orders
- 🔐 Secure user authentication (Login/Register)
- 📱 Fully responsive design
- 🧑‍💼 Admin dashboard for managing products and orders
- 💳 (Optional) Payment gateway integration (Stripe)

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose for ODM)
- JSON Web Tokens (JWT) for auth
- Bcrypt.js for password hashing
